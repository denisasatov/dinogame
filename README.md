# Dino Adventure

Клон классической игры Chrome Dino Run с поддержкой нескольких тем оформления.

## 🎮 Особенности

- **4 темы оформления**: тёмная, светлая, красная и неоновая
- Сохранение рекорда в localStorage
- AI-генерируемые мудрые цитаты при проигрыше
- Плавная анимация и отзывчивое управление
- Поддержка паузы

## 🚀 Запуск

Просто откройте файл `index.html` в браузере.

```bash
# Или используйте простой HTTP-сервер
python -m http.server 8000
# Затем откройте http://localhost:8000
```

## 🎨 Темы

| Тема | Описание |
|------|----------|
| 🌙 Тёмная | Классическая тёмная тема с приглушёнными тонами |
| ☀️ Светлая | Светлая тема для дневной игры |
| 🔥 Красная | Агрессивная красная тема |
| ⚡ Неоновая | Яркая неоновая тема с эффектом свечения |

## 🎯 Управление

| Клавиша | Действие |
|---------|----------|
| `Space` / `↑` / `W` | Прыжок |
| `Esc` | Пауза |

Также можно кликать мышью или тапать на экране для прыжка.

## 🦕 Геймплей

Управляйте динозавриком, прыгайте через кактусы и птиц. Со временем скорость увеличивается, делая игру сложнее.

## 💾 Сохранение

- Рекорд сохраняется автоматически в браузере
- Выбранная тема сохраняется между сессиями

## 🛠 Технологии

- HTML5 Canvas
- CSS Variables для тем
- Vanilla JavaScript (ES6+)
- Tailwind CSS
